Explorando a Programação Orientada a Objetos com Qt

Olá a todos! Hoje, quero compartilhar com vocês o emocionante projeto que desenvolvi na disciplina de Programação Avançada, sob a orientação do professor Agostinho Brito. Este projeto envolveu a aplicação prática de conceitos de programação orientada a objetos, utilizando a poderosa biblioteca Qt. Vamos explorar como criei três módulos interconectados para simular um sistema de aquisição e supervisão de dados em uma rede local, usando comunicação TCP/IP.

O projeto consistiu na criação de três programas de computador que funcionam em conjunto para simular um sistema de aquisição e supervisão de dados em uma rede local. Cada um desses programas desempenha um papel específico na operação do sistema.

1. O Servidor TCP/IP

O servidor é o núcleo do sistema. Ele aguarda conexões TCP na porta 1234 e responde aos comandos enviados pelos clientes. Este módulo já estava pronto e não precisou ser desenvolvido. Sua função é garantir a comunicação e coordenação entre os clientes produtores de dados e o cliente supervisor.

2. O Cliente Produtor de Dados

O cliente produtor de dados é responsável por gerar e enviar informações para o servidor. Ele pode se conectar à máquina em execução com o servidor na porta 1234 e enviar dados, juntamente com marcações de data/hora. Isso é fundamental para aquisição e armazenamento de informações no sistema.

3. O Cliente Supervisor de Dados

O cliente supervisor de dados atua como um observador do sistema. Ele também se conecta ao servidor na porta 1234, mas sua função é recuperar informações. Pode obter a lista dos clientes produtores de dados e recuperar listagens de dados produzidos por um desses clientes. Isso é essencial para supervisionar e analisar os dados gerados pelo sistema.

Desenvolver esses módulos foi um desafio empolgante. Utilizei os conceitos de programação orientada a objetos para criar classes e objetos que representavam os clientes produtores e supervisores. A biblioteca Qt provou ser uma escolha excelente, oferecendo ferramentas poderosas para criar interfaces gráficas e gerenciar a comunicação TCP/IP.

Este projeto me proporcionou uma sólida compreensão da programação orientada a objetos e a oportunidade de aplicar esses conhecimentos na prática, utilizando a biblioteca Qt. Agradeço ao professor Agostinho Brito por orientar e incentivar meu desenvolvimento nesse projeto desafiador. Espero que minha experiência inspire outros a explorar o mundo da programação orientada a objetos e a criação de aplicativos interconectados. A aprendizagem contínua é fundamental, e projetos como esse são uma maneira incrível de adquirir habilidades valiosas no campo da programação.
